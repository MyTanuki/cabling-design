# CCTV Cabling Designer

Web App สำหรับออกแบบระบบเครือข่ายกล้องวงจรปิด (CCTV) ตามหลักวิศวกรรม
โครงสร้างระบบ: **FTTx ONU → Distribution Switch → PoE Switch 8GE → CCTV** (อุปกรณ์หลัก Hikvision)

## วิธีเปิดใช้งาน

ต้องเปิดผ่าน HTTP server (ห้ามเปิดไฟล์ตรง ๆ เพราะ canvas จะ export ภาพไม่ได้):

```
npx --yes http-server -p 5179 -c-1 .
```

แล้วเปิด http://localhost:5179

## ขั้นตอนการใช้งาน

1. **อัปโหลดภาพสถานที่** — ลากไฟล์ภาพมาวาง หรือกดเลือกไฟล์ (มีปุ่มภาพตัวอย่าง `picture.jpg`)
2. **ตั้งค่ามาตราส่วน** — กด "วัดจากภาพ" คลิกจุดต้น–ปลายของแถบสเกลในภาพ แล้วกรอกระยะจริง (เมตร)
3. **วางจุดอุปกรณ์** — FTTx ONU (ดาวแดง), Distribution SW, PoE Switch 8GE, กล้อง CCTV (วงกลมเหลือง C)
4. **แนวเดินสาย** — ทำได้ 2 แบบ
   - **ออกแบบอัตโนมัติ**: วาดเส้น "ขอบอาคาร/แนวท่อ" บนภาพ (ดับเบิลคลิก/Esc จบเส้น) → กด "วาง DSW/PSW อัตโนมัติ" (จัดกลุ่มกล้อง ≤8 ตัว/สวิตช์ ยึดสวิตช์เข้าขอบอาคาร และเพิ่มจำนวนสวิตช์เองจนทุกกล้องอยู่ในระยะ CAT6 ตามเส้นทางเดินจริง) → กด "คำนวณแนวสายอัตโนมัติ" (หาเส้นทางสั้นสุดเลาะตามขอบอาคารด้วย Dijkstra โดยอุปกรณ์เชื่อมเข้าขอบอาคารได้หลายจุดเพื่อให้ได้เส้นสั้นที่สุด; ลิงก์สั้น ≤20 ม. เช่น ONU→DSW ตู้เดียวกัน ต่อตรง)
   - **แนวเดินสายทางเลือก**: กดปุ่ม 🔀 ท้ายแถวในตาราง → ระบบเสนอเส้นทางสำรอง (สูงสุด 3 เส้น) วาดเป็นเส้นประสีต่าง ๆ พร้อมระยะ — คลิกเส้นประบนภาพ หรือกด "ใช้เส้นนี้" ในรายการใต้ตารางเพื่อแทนที่แนวเดิม (Esc = ปิด)
   - **ลากเอง**: คลิกอุปกรณ์ต้นทาง → คลิกจุดหักมุมตามขอบอาคาร (บังคับ 90° ได้) → คลิกอุปกรณ์ปลายทาง
   - คลิกขวา = ย้อนจุดล่าสุด, `Esc` = ยกเลิก/จบเส้น, `Delete` = ลบรายการที่เลือก
5. **ผลการออกแบบ** — ตารางการเชื่อมต่อ (เลือกสภาพแวดล้อมต่อเส้น: ในร่ม/กลางแจ้ง/ฝังดิน), สรุปท่อร้อยสาย+อุปกรณ์ประกอบ, สรุปอุปกรณ์ Hikvision, ข้อแนะนำการติดตั้ง
   - **คลิกแถวในตาราง** = เลือกเส้นนั้นและเลื่อนภาพไปยังตำแหน่งบนแผนที่
   - **จัดลำดับรายการ** = กดปุ่ม ▲▼ หรือลากที่สัญลักษณ์ ⠿ หน้าแถว (ลำดับใหม่สะท้อนในรายงานด้วย)

## กติกาทางวิศวกรรมที่ระบบใช้

- **เดินสายตามแนวเท่านั้น:** เมื่อวาดแนวอาคาร/แนวท่อแล้ว สายอัตโนมัติจะวิ่งบนแนวเท่านั้น ไม่ตัดข้ามพื้นที่/อาคาร
  จุดติดตั้งที่ห่างแนว ≤ **10 ม.** จะแตก branch สั้นจากจุดใกล้สุดของแนวเดิม · จุดที่ไกลกว่าจะถูกข้ามพร้อมคำเตือนให้วาดแนวเพิ่ม
  (ยกเว้นคู่ที่ห่างกัน ≤10 ม. เช่น ONU→DSW ตู้เดียวกัน ต่อตรงแบบ patch ได้) และมีตัวตรวจเส้นที่ลากเอง/มีอยู่แล้วว่ามีช่วงออกนอกแนวหรือไม่
- สาย **LAN CAT6 (เส้นสีเหลือง)** ระยะไม่เกิน **180 ม.** — ช่วง 100–180 ม. จะเตือนให้ใช้โหมด Extended PoE
- ระยะเกิน 180 ม. เปลี่ยนเป็น **Fiber Optic (เส้นสีแดง)** อัตโนมัติ (ปรับเองได้ในตาราง)
  หากปลายทางเป็นกล้อง จะเตือนให้ใช้ Media Converter/SFP + จ่ายไฟกล้องแยก
- ความยาวสั่งซื้อ = ระยะตามแบบ ×1.10 + เผื่อปลายทาง 10 ม.
- **ท่อร่วม (วสท. 022001-22):** ระบบแตกทุกเส้นทางเป็นช่วงตามแนวเส้นตรงจริง แล้วนับสายที่เดินทับแนวเดียวกันจริง
  ก่อนคิดขนาดท่อต่อช่วง (fill ratio 53% / 31% / 40% สำหรับ 1 / 2 / ≥3 สาย) — เปิดดูได้ที่ตัวเลือก "ท่อร้อยสาย" บนแถบเครื่องมือ
  เลือกได้ 2 รูปแบบ: **ป้ายบนเส้น** (ชนิด/ขนาด/จำนวนสาย/ความยาว บนแนวท่อ) หรือ **กรอบชี้เส้น** (callout แบบแบบแปลนวิศวกรรม
  แยกจำนวนสายตามชนิด CAT6/Fiber) พร้อมสีตามขนาดท่อ + ขีดจำนวนสายที่ทับกัน และดูสรุปในหัวข้อ "ท่อร้อยสายและอุปกรณ์ประกอบ" ที่แยกตามขนาดท่อจริง
  แนวเดินสายอัตโนมัติจะเลือกเดินร่วมท่อเดิมก่อนถ้าคุ้ม เพื่อลดจำนวนท่อที่ต้องเดินแยก
- ชนิดท่อตามสภาพแวดล้อม: ในร่ม **EMT** · นอกอาคารกันน้ำ/ทน UV **IMC / uPVC** · ฝังดิน **HDPE/PE** (ลึก ≥60 ซม.)
  พร้อมคำนวณอุปกรณ์ประกอบ: ข้อต่อ, คอนเน็คเตอร์, ข้องอ, แคล้มป์, Pull Box / Handhole, เทปเตือนใต้ดิน, ทรายรอง
- ตรวจสอบอัตโนมัติ: กล้องต่อ PoE Switch เกิน 8 พอร์ต, กล้องที่ยังไม่ได้ลากสาย

## ผลลัพธ์

- **Export PNG** — ภาพแบบแปลนพร้อมแนวเดินสาย ป้ายระยะ และคำอธิบายสัญลักษณ์ ที่ความละเอียดเต็มของภาพต้นฉบับ
- **พิมพ์รายงาน** — รายงาน 5 ส่วน: แบบแปลน / ตารางการเชื่อมต่อจุดต่อจุด (ชนิดสาย ระยะ ท่อร้อยสาย) / สรุปท่อร้อยสายและอุปกรณ์ประกอบแยกตามสภาพแวดล้อม / สรุปอุปกรณ์ Hikvision / ข้อแนะนำการติดตั้งตามมาตรฐาน (TIA-568/569/606/607)
- งานออกแบบบันทึกอัตโนมัติใน localStorage ของเบราว์เซอร์ (ปิด-เปิดใหม่งานยังอยู่)
